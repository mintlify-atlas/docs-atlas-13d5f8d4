## Required Packages

This project requires several Julia packages for data processing, linear algebra, machine learning, and visualization.

## Quick Installation

You can install all required packages at once using the following command:

```julia
import Pkg
Pkg.add("LinearAlgebra")
Pkg.add("Plots")
Pkg.add("Statistics")
Pkg.add("CSV")
Pkg.add("DataFrames")
Pkg.add("Random")
Pkg.add("Clustering")
Pkg.add("PlotlyJS")
```

<Note>
  This installation command is taken directly from the project notebook (cell 1).
</Note>

## Verification

Verify that all packages are installed correctly by loading them:

```julia
using LinearAlgebra
using Plots
using Statistics
using CSV, DataFrames
using Random
using Clustering
```

If all packages load without errors, your environment is ready!

<Warning>
  Package installation may take several minutes, especially for Plots and PlotlyJS which have many dependencies.
</Warning>

## Troubleshooting

### Package Installation Fails

If you encounter errors during installation:

```julia
# Update the package registry
using Pkg
Pkg.update()

# Then try installing again
Pkg.add("PackageName")
```

### Precompilation Errors

If packages fail to precompile:

```julia
using Pkg
Pkg.build()  # Rebuild all packages
```
