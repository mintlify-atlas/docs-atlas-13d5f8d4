## Overview

This research project uses machine learning techniques to analyze online education logs from Riiid, an AI EdTech company based in South Korea. By identifying student performance patterns and predicting success rates, this work enables personalized educational interventions that meet students exactly where they're struggling.

## Key Features

- **Performance Segmentation**: K-Means clustering with 20 clusters identifies distinct student performance groups based on accuracy, standard deviation, and number of attempts
- **Predictive Modeling**: Regularized least squares model achieves ~80.7% accuracy in predicting student success
- **Advanced Features**: Engineered metrics including mean accuracy, consistency trends, performance trends, and tutorial explanation effectiveness
- **Comprehensive Evaluation**: Confusion matrix analysis with precision, accuracy, and detailed performance metrics
- **Julia Implementation**: Built with DataFrames.jl, Clustering.jl, LinearAlgebra.jl, and PlotlyJS.jl

## Quick Start

<Steps>
  <Step title="Set up Julia environment">
    Install Julia 1.11.2 and required dependencies
    
    ```julia
    import Pkg
    Pkg.add(["LinearAlgebra", "Statistics", "CSV", "DataFrames", "Random", "Clustering", "PlotlyJS"])
    ```
  </Step>
  
  <Step title="Prepare your data">
    Load and preprocess RIIID education logs
    
    ```julia
    using CSV, DataFrames
    
    df = DataFrame()
    for chunk in CSV.Chunks("train.csv"; ntasks=10)
        df_chunk = DataFrame(chunk)
        df_chunk = filter(row -> rand() < 0.01, df_chunk)
        filter!(row -> row.content_type_id == 0, df_chunk)
        append!(df, df_chunk)
    end
    ```
  </Step>
  
  <Step title="Train models">
    Engineer features and train both clustering and regression models

    # Train regularized least squares

## Results Snapshot

The regularized least squares model with λ=0.01 achieved:

- **Accuracy**: ~80.7% on test data
- **Precision**: 100% in predicting student success
- **Feature Engineering**: 11 engineered features including mean accuracy, performance trends, explanation impact, and time-accuracy correlation
- **Clustering**: 20 distinct student performance groups identified through K-Means

These findings demonstrate that predictive modeling with performance-based clustering can support the development of personalized education strategies, enabling educators to reliably identify students who need intervention.
